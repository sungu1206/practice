<a></a> 이 태그는 링크 요소이다. 이 요소의 가장 중요한 속성은 링크의 목적지를 나타내는 href 속성이다.
<abbr></abbr> 이 태그는 약어에 대한 정의를 표시한다. ex) WWW , KSG (link)
<address></address> 이 태그는 문서나 문서의 저자 / 소유자의 연락처 정보를 정의합니다.(이메일 주소, URL, 실제 주소, 전화 번호, 소셜 미디어 핸들 등..) (link)
<area> </area> 태그는 이미지 맵 (이미지 맵 클릭 가능한 영역을 갖는 화상이다.) 내부의 영역을 정의한다. (link)
<article> </article> 이 태그는 독립적인 콘텐츠를 포함한다.(뉴스, 기사, 블로그 등..) (link)
<aside> </aside> 이 태그는 본문(주요 컨텐츠)와 직접적인 관련은 없으나, 간접적으로 관련된 추가 정보를 제공한다.(사이드바, 배너광고, 위젯 등..) (link)
<audio> </audio> 이 태그는 음악이나 다른 오디오 스트림으로서, 문서에 포함 된 음성 콘텐츠를 사용한다. (link)
